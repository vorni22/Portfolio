Projects

A selection of things I've built and worked on.

Train Island

A 3D train simulation game built with OpenGL. Navigate a locomotive across a procedurally generated island with Perlin noise terrain, collect resources from stations, and deliver them before time runs out. Features Phong shading, tunnels with occluded rendering, a minimap, and a mission system.

  • C++
  • OpenGL
  • GLSL
  • CMake

YetAnotherTown

A work-in-progress real-time traffic simulation engine featuring an expandable multithreaded architecture, LOD-based terrain rendering, terraforming tools (flatten, smooth, raise, slope), and a cubic Bézier-based road-drawing system that auto-integrates into the road network with computed intersections.

  • C++
  • OpenGL
  • Multithreading

PyParser

A minimalistic parser built entirely from scratch in Python for educational purposes. Includes a custom regex engine, lexer, and grammar-driven parser — no external libraries used.

  • Python
  • Parsing
  • Regex
  • Compilers

PyKingdom Group Project

A 4X strategy game inspired by the Civilization series, playable in hot-seat multiplayer. Features a hexagonal map with procedural generation, OpenGL rendering with custom GLSL shaders, 3D models built in Blender, and a city/unit management system — all powered by Python and PyGame.

Other contributors: Ursu Vasile · Condorache Alexandru
  • Python
  • OpenGL
  • PyGame
  • GLSL

Asynchronous C++ Logger

A lightweight asynchronous logging system in C++. The main process writes log messages to a shared memory buffer, while a separate logger process reads from that buffer and writes to a log file independently, keeping the main application non-blocking.

  • C++
  • Shared Memory
  • IPC
  • Makefile
